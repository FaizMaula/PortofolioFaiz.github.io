Project Strategi Algoritma

The Knapsack problem is a problem where a person is faced with the optimization challenge of selecting items that can fit into a knapsack that has limitations in capacity and space. In this study, we applied the Knapsack Problem to the Fighters Selector application using the Brute Force algorithm and the Greedy algorithm. The purpose of this research is to compare which algorithm provides the most optimal solution between the Brute Force and Greedy algorithms. The results show that the Brute Force algorithm offers a more optimal solution compared to the Greedy algorithm.

Klasifikasi Kurma

Developed a machine learning model to classify dates based on their physical attributes (e.g., color, size) using the K-Nearest Neighbors (K-NN) algorithm. The model predicts the class of a date by analyzing its similarity to previously labeled data points. This project demonstrates practical application of supervised learning techniques for classification tasks.
